# Burrow

A browser for the [Gopher](https://en.wikipedia.org/wiki/Gopher_(protocol)) protocol, running in the browser.

**Demo:** https://burrow.din.gy/

## Running

This project requires a recent Node.js and a browser. It has 0 dependencies.
Just clone and run:

```bash
node server.mjs
```
